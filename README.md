# Conversor de Monedas en C

## Descripción
Este programa se realizo en el lenguaje de C
## Prerrequisitos
- GCC (compilador de C)
- Linux o Ubuntu
- Terminal

## Pasos para ejecutar

1. Compilar el programa

gcc conversor.c -o conversor

2. Ejecutar

./conversor
